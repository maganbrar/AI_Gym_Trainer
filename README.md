# 🏋️ AI Gym Trainer - AI Backend

AI Gym Trainer is an AI-powered virtual fitness assistant built using Python, Streamlit, OpenCV, and AI/ML technologies.

This backend application provides real-time exercise analysis, AI posture detection, workout interaction, and intelligent fitness assistance using computer vision and AI models.

🌐 Live AI App: https://ai-gym-trainer-uj9g.onrender.com/

---

# 🚀 Features

- 🤖 AI-powered virtual gym trainer
- 🧠 AI posture detection
- 📹 Real-time exercise analysis
- 💪 Exercise monitoring using computer vision
- 🎯 Smart workout interaction

# ⚡ Project Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/maganbrar/AI_Gym_Trainer.git
```

---

## 2️⃣ Open Project Directory

```bash
cd AI_Gym_Trainer
```

---

# ⚡ Create Virtual Environment Using UV

## Install UV

```bash
pip install uv
```

---

## Verify UV Installation

```bash
uv --version
```

---

## Create Virtual Environment

```bash
uv venv
```

---

## Activate Virtual Environment

### Windows

```bash
.venv\Scripts\activate
```

### Linux / Mac

```bash
source .venv/bin/activate
```

---

# 📦 Install Dependencies

```bash
uv pip install -r requirements.txt
```

---

# 🔑 Setup Environment Variables

Create a `.env` file inside the project root.

Example:

```env
GROQ_API_KEY=your_api_key_here
```

---

# ▶️ Run Application

```bash
uv run streamlit run main.py
```
